✦ BFS Algorithm Explanation

1
Start from the user: Begin at the selected user node in the graph
2
Level 1 traversal: Visit all direct friends (immediate connections)
3
Level 2 traversal: Visit friends of friends (potential recommendations)
4
Filter results: Exclude users already connected to the current user
5
Rank by mutual friends: Sort recommendations by number of mutual connections
Time Complexity: O(V + E) | Space Complexity: O(V)

Where V = vertices (users) and E = edges (connections)
