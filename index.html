🔬 DSA VISUALIZER — Live Java Graph Calls

Graph Traversal Explorer

Run BFS and DFS on the live Java Graph. Results come from Graph.java on the Spring Boot server.

🌐 BFS Traversal

Algorithm: Queue-based. Visits nodes level-by-level.
Used for: Friend-of-friend recommendations.

🔭 DFS Traversal

Algorithm: Stack-based. Explores as deep as possible first.
Used for: Deep network exploration.

📊 FULL DATA FLOW — HOW EVERY REQUEST WORKS

Browser fetch() ──→ HTTP POST /api/friends/suggest ──→ Spring @RestController
                                                      ↓
@Autowired SocialMediaService ──→ RedBlackTree.getValue(username) O(log n)
                                                      ↓
Graph.recommendFriends(userId) ── BFS depth-2, mutual-count sort ── O(V+E)
                                                      ↓
JSON { success, data: [...] } ──→ Frontend renders suggestion cards
